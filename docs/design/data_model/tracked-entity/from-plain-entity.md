# from-plain-entity

**Data Model → `data_model/tracked-entity/`**

---

## Purpose

`From<PlainEntity>` converts a fully-populated plain entity into its tracked counterpart. Generated by the derive macro. Each field is wrapped in `Arc<TrackedField<T>>` with its `OnceLock` pre-seeded with the plain entity's value and `dirty = false` — all fields loaded, all fields clean.

---

## Generated Implementation

```rust
impl From<Role> for TrackedRole {
    fn from(role: Role) -> Self {
        TrackedRole {
            entity_ref:  role.entity_ref,
            name:        Arc::new(TrackedField { value: OnceLock::from(role.name),        dirty: false }),
            purpose:     Arc::new(TrackedField { value: OnceLock::from(role.purpose),     dirty: false }),
            traits:      Arc::new(TrackedField { value: OnceLock::from(role.traits),      dirty: false }),
            extensions:  Arc::new(TrackedField { value: OnceLock::from(role.extensions),  dirty: false }),
        }
    }
}
```

---

## When Used

`EntityClient::insert()` sends the plain entity to `EntityServer`. The server calls `From` internally to convert it to tracked form before inserting into `entities` and adding the ref to `added`. The conversion happens inside `EntityServer`, not on the caller's side.

---

## Invariants

- All fields' `OnceLock` are initialized — the plain entity is always fully populated
- All fields have `dirty = false` — the entity is considered clean at insert time; it is in `added` and will be written at persist, but its individual fields are not dirty
- `entity_ref` is moved directly — no wrapping needed
